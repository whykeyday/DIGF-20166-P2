## 1. 项目概念更新（Concept）
- 主题方向：**Take a Breath / Gentle Reminder**
- 我想感知的体验：
  1) **久坐/不动**（身体状态：长时间静止）  
  2) **紧张时无意识抠袖子**（身体动作：重复、较强的压力动作）  
  3) **环境噪音**（周围环境：声音大时更容易紧张）
- 作品意图：
  - 不是"命令式纠正"，而是用 **轻柔、低打扰** 的方式，提醒我回到呼吸、放松手部、或起身活动。
  - 批判性角度（草案）：效率/学习压力下的自我规训与身体信号被忽略；用"可穿戴的温柔介入"把注意力拉回身体与当下。

## 2. 供电方案（Power）✅ 已确认
### 2.1 电池类型
- 电池：**EBL 1100mAh Ni-MH Rechargeable AA Batteries**
- Ni-MH 充电电池为 **1.2V/节**
- 使用 **3 节串联电池盒**：
  - 标称电压约 **3.6V**
  - 满电时可能到 **约 4.0–4.2V**（仍在相对合理范围）

### 2.2 连接方式
- 供电口：使用 CPX 板上的 **白色 2-pin 电池接口（JST）**
- 接线极性：
  - 电池盒 **红线 = +** → CPX 电池口 **正极**
  - 电池盒 **黑线 = −** → CPX 电池口 **负极 / GND**

### 2.3 风险规避（结论）
- ✅ Ni-MH 3 节串联 **可以用且更稳定**（比纽扣电池强）
- ❌ 不使用 3 节 1.5V 碱性电池（总电压可能 4.5–4.8V，存在过压风险）

### 2.4 若出现不稳定（排查清单）
- 检查电池是否充满、接触是否牢固
- 若 NeoPixel 灯效过亮导致瞬间电流过大：
  - 降低亮度
  - 减少同时点亮颗数
  - 减少高频闪烁/爆闪

## 3. 传感器与引脚（Sensors & Pins）
### 3.1 使用的传感器（满足作业要求）
- 至少两个模拟传感器：
  1) **DIY Pressure Sensor（外接模拟传感器）** ✅（满足"至少一个不是 CPX 板载"）
  2) **CPX 板载 Sound Sensor（MIC）** ✅
  3)（可选）CPX 板载 Motion（Accelerometer）用于久坐/不动判断 ✅

> 说明：CPX 板载传感器也属于 allowed list；DIY pressure sensor 属于 variable resistor + voltage divider。

### 3.2 DIY Pressure Sensor 的分压电路（Voltage Divider）
- 目的：把"可变电阻（压力）"变成 CPX 可读的模拟电压 Vout
- 结构（标准分压）：
  - **CPX 3.3V → Rvar（DIY pressure sensor） → Vout（节点） → Rfixed（固定电阻） → CPX GND**
  - **Vout（节点）→ A2（Analog In）**

### 3.3 引脚选择说明（A1/A4/A5/A2）
- A1/A2/A4/A5 都是模拟输入脚，本质差别不大
- 我目前使用：**A2 读取压力传感器 Vout**
- 如果未来做左右手两片压力传感器，可用：
  - 左袖 Vout → A5
  - 右袖 Vout → A4

## 4. 执行器（Actuators）与模式（Modes）
> 作业要求：至少 **3 个 response modes**（由灯或声音体现）

### 4.1 执行器选择（符合限制）
- 主要：**NeoPixel LED（CPX 板载灯环）**
- 可选：**tone（蜂鸣/音调）**（如果需要更明确提醒）

### 4.2 计划的 3 个模式（草案，可调参）
1) **Mode 1：Normal / Calm**
   - 条件：压力动作不频繁；环境噪音不高；非久坐提醒状态
   - 输出：灯光轻微"呼吸"或低频缓慢变化（不打扰）

2) **Mode 2：Take a Breath（紧张/抠袖子触发）**
   - 条件：压力值超过阈值，且在短时间内出现多次（频繁抠袖子）
   - 输出：
     - 灯光更明确的呼吸节奏（例如吸气 4s / 呼气 6s，循环 3 次）
     - （可选）很轻的提示音，不做"报警音"

3) **Mode 3：Noise-weighted / Overstimulated（噪音加权）**
   - 条件：环境噪音高 + 压力动作频繁（更容易紧张的组合）
   - 输出：
     - 灯光节奏更"可读"（更明显但仍温柔）
     - （可选）比 Mode2 更容易被注意到的低频提示音

>（可选 Mode 4：Move reminder）  
- 条件：加速度变化长期很小（久坐/长时间不动）  
- 输出：灯光慢闪提醒起身/伸展（尽量不羞辱式、不强迫）

## 5. 测试计划（Testing Plan）
### 5.1 压力传感器（DIY patch）测试
- 目标：找到稳定阈值，减少误触发
- 方法：
  - 串口输出 pressure raw value（A2）
  - 测三种状态：
    1) 手自然放松/不碰袖口（baseline）
    2) 轻触/轻抠（low）
    3) 用力抠（high）
  - 记录每种状态的数值范围，选阈值（threshold）与回差（hysteresis）

### 5.2 声音（MIC）测试
- 目标：区分 "安静/正常/嘈杂"
- 方法：
  - 记录安静房间 baseline
  - 在教室/咖啡店等环境记录 noisy 值
  - 设置两个阈值：low / high（并加入平滑）

### 5.3 模式切换测试
- 检查：
  - 模式是否来回抖动（需要 hysteresis 或 cooldown）
  - LED 是否足够表达差异（可读性）
  - 在佩戴状态下，线是否拉扯、接触是否松动
